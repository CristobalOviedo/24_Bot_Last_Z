# Plantillas de construcción

Coloca aquí los recortes PNG asociados a la tarea `construction`. Cada archivo debe conservar la resolución nativa (540p) y el fondo exacto capturado desde BlueStacks.

## Elementos requeridos

| Nombre de template | Descripción del recorte |
| --- | --- |
| `construction_icon.png` | Icono flotante que indica que hay una mejora disponible. |
| `construction_hq_panel.png` | Encabezado del panel del Cuartel General. |
| `construction_upgrade_button.png` | Botón azul "Mejorar" habilitado dentro del HQ. |
| `construction_upgrade_button_disabled.png` | Variante gris del botón cuando faltan requisitos. |
| `construction_go_button.png` | Botón "Ir" que lleva al edificio requisito. |
| `construction_requirement_upgrade.png` | Botón azul "Mejorar" del edificio requisito. |
| `construction_resource_shortage.png` | Ventana/popup que avisa falta de recursos. |
| `construction_resource_autofill_button.png` | Botón "Rellenar automáticamente" dentro del popup de recursos. |
| `construction_resource_confirm_button.png` | Botón "Aceptar" que confirma el rellenado. |
| `construction_help_button.png` | Botón para solicitar ayuda de la alianza tras iniciar la construcción. |
| `construction_timer_panel.png` | Header del panel temporal que se abre al tocar el icono de construcción para leer el timer. |
| `construction_level_up_overlay.png` | Ventana que aparece al subir de nivel un edificio y que debe cerrarse antes de continuar. |

Guarda cada imagen siguiendo exactamente los nombres anteriores para que `config/farms.yaml` las resuelva sin ajustes adicionales.
